## keys

Zoom In  Crtl+Z

Zoom out Crtl+X

normal size Crtl+A

change line width Q

change shape W

Remove Box R

Next Image ->

prev Image <-

Copy Box Crtl+C

paste Box Crtl+V

Save Crtl+S

Delete Image Crtl+I
